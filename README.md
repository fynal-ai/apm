APM (Agent Package Manager)

# Quick

1. download docker-compose.yml and start APM Server
   ```sh
   docker compose -f docker-compose.yml up
   ```
2. install APM CLI
   ```sh
   pnpm install -g @jobsimi/apm
   apm --help
   ```
